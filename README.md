# Personal Website

This is my personal website hosted on GitHub Pages at [joebarranquilla.github.io](https://joebarranquilla.github.io)

## Features

- Clean and modern design
- Fully responsive (mobile-friendly)
- Easy to customize
- Built with HTML and CSS

## Customization

To make this website your own:

1. Edit `index.html` to update your content (name, bio, projects, links)
2. Modify `styles.css` to change colors, fonts, or layout
3. Update project cards with your actual projects
4. Add your social media links in the contact section

## How to Deploy

1. Create a new repository on GitHub named `joebarranquilla.github.io`
2. Push this code to that repository
3. Your site will be live at `https://joebarranquilla.github.io` within a few minutes

## GitHub Pages Setup

To push to GitHub:

```bash
git add .
git commit -m "Initial commit: personal website"
git branch -M main
git remote add origin https://github.com/joebarranquilla/joebarranquilla.github.io.git
git push -u origin main
```

That's it! Your website is now live.
